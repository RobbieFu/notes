#单例模式
##饿汉式
```
class Singleton {
  private static Singleton mInstance = new Singleton();
  
  private Singleton() {
  }
  
  public static Singleton getInstance() {
    return mInstance;
  }
  
}
```
##懒汉式
```
class Singleton {
  private static Singleton mInstance = null;
  
  private Singleton() {
  }
  
  public static synchronized Singleton getInstance() {
    if(mInstance == null) {
      mInstance = new Singleton();
    }
    
    return mInstance;
  }
  
}
```
